# Data graph visualizer 

##### Requirements:
Python 3.8.10

##### Prepare environment:

```
python3 -m venv libs
. ./libs/bin/activate
pip3 install -r requirements.txt
```

##### Run:
! If you have activated (during installed or whatever) libs earlier, may skip:
```
. ./libs/bin/activate
```
Actual run:
```
streamlit run app.py
```
